# AutoShorts.ai 🎬

> AI-powered faceless video generator for TikTok & YouTube Shorts

## 🚀 Live App
👉 **[Open AutoShorts.ai](https://YOUR-USERNAME.github.io/autoshorts/app.html)**

## ✨ Features
- 🎨 **17 Art Styles** — LEGO, Anime, Disney Toon, Film Noir, 3D Toon, Watercolor & more
- 🤖 **AI Script Writing** — Claude AI writes viral scripts automatically
- 🖼️ **AI Image Generation** — Real images via kie.ai Z-Image API
- 🎙️ **Text-to-Speech** — Browser voiceover with multiple voice styles
- 🎵 **Background Music** — Synthesized cinematic audio (Web Audio API)
- 📱 **TikTok Preview** — Phone mockup with word-by-word captions
- 🔒 **Secure** — API key never stored, rate-limited, endpoint-locked

## 🛠️ Tech Stack
- Pure HTML/CSS/JS — zero dependencies
- [kie.ai Z-Image API](https://kie.ai/z-image) for image generation
- Claude API (Anthropic) for script writing
- Web Audio API for music synthesis
- Web Speech API for voiceover
